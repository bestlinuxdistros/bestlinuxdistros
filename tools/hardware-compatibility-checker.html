Hardware Planning

Linux Hardware Compatibility Checker

Use this worksheet to compare your laptop or server specs against the requirements published on each distro detail page. It keeps you honest before you flash an ISO.

1. Capture your current hardware

Component Your Spec Notes
CPU e.g., Intel i5-8250U (4C/8T) Record generation and whether virtualization is enabled.
RAM 8 GB DDR4 Check if slots are upgradeable.
Storage 512 GB NVMe (unused 220 GB) Note free space for dual-boot.
GPU Nvidia MX150 Flag hybrid graphics for distros like Pop!_OS.

2. Compare against distro requirements

The easiest way is to open the distro detail page and scroll to Hardware requirements. Start with these averages:

Distro Minimum Recommended
Ubuntu 2 cores / 2 GB RAM / 20 GB disk 4 cores / 4 GB RAM / 40 GB SSD
Linux Mint 2 GB RAM / 20 GB disk 4 GB RAM / 100 GB disk for comfort
Kali 2 cores / 2 GB RAM 4 cores / 8 GB RAM when running VMs
Clear Linux AVX2-capable CPU / 8 GB RAM Modern Intel CPU / 16+ GB RAM / NVMe storage

3. Plan upgrades or distro swaps

  • If RAM is below requirements, prioritize a lightweight distro such as MX Linux or Peppermint.
  • If your GPU is proprietary-only, choose distros with driver toggles (Pop!_OS, Garuda Dragonized).
  • Use the distro decision guide to validate the rest of your shortlist.

4. Checklist before you install

  1. BIOS/UEFI updated and Secure Boot status documented.
  2. Backups complete (Timeshift, Veeam, or rsync).
  3. Free space carved out for the install (at least 30 GB recommended).
  4. Network adapters confirmed in Linux live session.

Once everything is green, boot the ISO and follow the steps in the Beginner guide.
